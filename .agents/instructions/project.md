# Project Overview

This is a GitHub base repository template (`@duncan3142org/base-template`).
It provides a starting point for new repositories in the Duncan3142Org organization.

## Tech Stack

- Node.js (>= 24) / npm (>= 11)
- [Mise](https://mise.jdx.dev/) for task definition / execution and cli tooling
- [Dev Containers](https://containers.dev/) for the development environment
- GitHub Actions for CI/CD
