# Workflows

## Development

- `MISE_ENV=dev` is set within the dev container, DO NOT MODIFY IT
- Use `mise run <task>` to execute dev tasks
- Use `mise run <task> --help` to print task help

## Formatting

- Run `mise run format check` to check formatting
- Run `mise run format write` to fix formatting

## Assests

- Run `mise run assets` to generate assets from source code

## Linting

- Run `mise run lint check` to check linting
- Run `mise run lint fix` to fix linting

## Testing

- Run `mise run test` for tests
