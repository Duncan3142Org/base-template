# Repository Structure

- `.github/` - GitHub Actions workflows, repository environment terraform config
- `.mise/` - Mise task and tool configuration
- `tasks/` - Mise task scripts
- `src/` - Repo source code, unit & contract tests
- `test/` - Integration tests
- `.devcontainer/` - Dev Container configuration.
  - `↳/devcontainer/.config/mise/config.toml` - Dev Container tool configuration
- `CHANGELOG.md` - Auto generated changelog
- `package.json` - Project metadata and dependencies
