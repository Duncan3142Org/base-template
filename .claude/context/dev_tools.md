# Dev Tools

- [Mise](https://mise.jdx.dev/) for task definition / execution and cli tooling
- [Dev Containers](https://containers.dev/) for the development environment
- Prettier for formatting, config in `.prettierrc.js`
- ESLint 9+ for linting, config in `eslint.config.js`
- Typescript for type safety
  - Main config in `tsconfig.json`
  - Sub project `tsconfig.*.json` files per file family, i.e. `tsconfig.assets.json` for built assets.
