# Workflows

- Use `mise run <task>` to execute dev tasks
- Use `mise run <task> --help` to print task help

## Clean

- Run `mise run clean` to clean cache files & built assets

## Formatting

- Run `mise run format` to check formatting
- Run `mise run format --mode write` to fix formatting

## Assests

- Run `mise run assets` to generate assets from source code

## Linting

- Run `mise run lint` to check linting
- Run `mise run lint --fix` to fix linting

## Testing

- Run `mise run test` for tests
